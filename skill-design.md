# 运动康复馆开馆助手 — 产品分析与 Skill 设计文档

## 一、产品定位与核心目的

### 这个网站是什么？

**"运动康复馆开馆助手"** 是健衡医疗（A&B Physiotherapy / Theratools）运营的 B2B SaaS 平台。它既是一个**获客漏斗**（为设备公司引流），也是一个**真实的商业规划工具**（为开店者生成完整方案）。

### 核心目的

> **帮助运动康复领域的创业者（投资人或康复师），用 6-8 分钟回答一组结构化问题，获得一份包含设备报价、装修指导、人员配置、盈亏分析、60 天开业时间表的完整开馆方案。**

### 一句话价值主张

**"先回答一些简单问题，再决定开什么馆"** —— 用数据驱动的诊断取代拍脑袋决策。

## 二、它为用户带来什么？

### 对创业者（客户）的价值

| 维度 | 具体价值 |
|------|---------|
| **认知校准** | 10 维能力评估（专业 5 维 + 经营 5 维），让用户清楚自己处于"起步型 / 流量增长型 / 专家型 / 系统型"哪个象限 |
| **定位匹配** | 根据能力得分 + 场地条件，自动推荐最合适的经营原型：社区轻量店 / 商圈标准店 / 医健协同店 |
| **设备方案** | 从 14+ 款专业设备中按场地、预算、服务类型自动匹配，生成基础版 / 标准版 / 升级版三档报价 |
| **财务模型** | 盈亏平衡分析：固定成本、变动成本率、贡献毛利率、安全营业额、产能利用率、90 天现金储备 |
| **落地执行** | 60 天开业甘特图（12 项任务）、10 项证照清单（含责任人、时间节点、所需材料）、5 个岗位的职责与 KPI |
| **装修指导** | 功能区面积配比、三线动线规划、装修优先级建议 |
| **可导出** | 一键生成 9 页 Excel 报表，可直接用于内部决策或与合伙人/投资人沟通 |

### 对平台运营方（健衡）的价值

| 维度 | 具体价值 |
|------|---------|
| **精准获客** | 通过诊断过程自然收集客户的能力画像、场地信息、预算范围 |
| **销售赋能** | 自动分配顾问、生成报价、追踪跟进，形成 CRM 闭环 |
| **设备推广** | 方案中嵌入自有设备产品线的精准推荐 |
| **内容沉淀** | 每个案例的诊断数据和成交状态形成知识库 |

## 四、评估体系详解

### 4.1 能力评估（10 维度，每维 0-100 分）

**专业维度（权重：技术分）**

| 维度 | 权重 | 评估内容 |
|------|------|---------|
| 专业背景 | 25% | 学历、培训经历、从业年限 |
| 评估与干预能力 | 25% | 能否独立完成评估、制定方案、执行治疗 |
| 风险认知 | 20% | 对禁忌症、转诊边界、医疗合规的理解 |
| 案例积累 | 15% | 接诊量、典型病例、效果记录 |
| 带教与团队 | 15% | 能否培训新人、建设团队 |

**经营维度（权重：商业分，等权）**

| 维度 | 权重 | 评估内容 |
|------|------|---------|
| 客户资源 | 20% | 是否有现成客户池 |
| 内容获取 | 20% | 能否持续产出专业内容（科普、案例） |
| 销售转化 | 20% | 从线索到成交的能力 |
| 管理能力 | 20% | 团队管理、流程管理、财务意识 |
| 资金与团队 | 20% | 启动资金充足度、核心团队完整度 |

### 4.2 象限分类

```
            商业能力 ≥ 60
                │
    ┌───────────┼───────────┐
    │  专家型    │  系统型    │
    │ (技术强,   │ (技术强,   │
    │  商业弱)   │  商业强)   │
技术 ├───────────┼───────────┤
能力 │  起步型    │ 流量增长型 │
≥ 60 │ (技术弱,   │ (技术弱,   │
    │  商业弱)   │  商业强)   │
    └───────────┼───────────┘
                │
            商业能力 < 60
```

### 4.3 经营原型推荐逻辑

| 原型 | 条件 | 适合场景 |
|------|------|---------|
| **医健协同店** | 技术分 ≥ 60 + 风险认知 ≥ 65 + (目标客群含术后 或 商业分 ≥ 70) + 面积 120-250㎡ | 有医疗背景的高端专业店 |
| **商圈标准店** | 面积 ≥ 150㎡ + 预算 ≥ 50 万 + 商业分 ≥ 55 | 有资金实力的商业化运营 |
| **社区轻量店** | 以上均不满足时的默认选项 | 轻启动、低风险试错 |

### 4.4 准备度评估（5 维度）

资金充足度、资源匹配度、参与深度预期、回报预期合理性、风险承受意愿 → 综合得分 < 60 准备不足 / 60-79 有条件推进 / ≥ 80 准备充分

## 六、封装为 Skill 的设计建议

### 6.1 Skill 名称

`rehab-studio-advisor`（运动康复馆开馆顾问）

### 6.2 Skill 触发条件

当用户提到以下关键词时触发：
- "我想开一家运动康复馆/工作室"
- "帮我评估一下适不适合开康复馆"
- "开康复馆需要什么条件"
- "运动康复创业"
- "康复工作室开店"
- "开馆评估" / "开店评估"

### 6.3 Skill 工作流设计

```
Phase 1: 身份识别
  → 询问：你是投资人还是康复师创业者？
  → 根据身份选择后续评估维度

Phase 2: 能力评估（10 题）
  → 专业维度 5 题（每题 4 个预设选项 + 自由填写）
  → 经营维度 5 题（每题 4 个预设选项 + 自由填写）
  → 每题给出评分依据说明

Phase 3: 场地与投入
  → 城市、面积、预算、租金模式、月租

Phase 4: 客群与团队
  → 目标客群（多选）、服务类型（多选）、团队状态、开业时间

Phase 5: 诊断与输出
  → 计算技术分、商业分、准备度
  → 确定象限定位
  → 推荐经营原型
  → 给出定制化建议清单
  → 生成结构化报告（Markdown 格式）
```

## 七、差异化建议要点

### 针对不同象限的差异化建议

**起步型（技术弱 + 商业弱）：**
- 建议先提升专业能力再开店，或寻找技术合伙人
- 推荐社区轻量店，低成本试错
- 重点投入个人品牌建设和内容输出
- 警示：不要在能力不足时盲目投入大量资金

**专家型（技术强 + 商业弱）：**
- 技术是核心优势，但商业能力是瓶颈
- 建议寻找商业合伙人或雇佣运营人员
- 推荐医健协同店（如有医疗背景）或社区轻量店
- 重点：不要用技术思维替代商业思维

**流量增长型（技术弱 + 商业强）：**
- 商业能力是优势，但技术短板可能影响口碑
- 建议加强技术团队建设，或聚焦引流/复购等非核心技术服务
- 推荐商圈标准店（如有资金实力）或社区轻量店
- 重点：确保服务质量，避免"流量陷阱"

**系统型（技术强 + 商业强）：**
- 最理想的组合，可以考虑更大的投入
- 推荐商圈标准店或医健协同店
- 重点：标准化复制、团队扩张、品牌建设

## 八、注意事项与免责

1. 本评估基于用户自述信息，可能存在偏差
2. 建议结合实地考察、市场调研、财务测算综合决策
