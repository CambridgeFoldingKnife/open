# 运动康复馆开馆助手 · 交付说明

## 已交付

- 微信小程序：手机号授权入口、开馆问卷、草稿恢复、准备度评分、审核进度和发布后方案。
- 管理后台：项目看板、顾问分配、方案报价编辑、管理员审核发布、馆型及产品课程维护。
- 服务端：角色权限、六阶段状态机、规则匹配、客户数据隔离、版本与审计、七工作表 Excel 导出。
- 数据存储：无配置时使用演示数据；配置 `DATABASE_URL` 后使用 PostgreSQL 持久化。

## 本地运行

在项目根目录执行：

```bash
npm run dev
```

管理后台地址：`http://localhost:5173`

微信小程序构建：

```bash
npm run dev:mini
```

随后使用微信开发者工具导入 `apps/mini`。

## 正式上线前配置

1. 将 `apps/mini/project.config.json` 中的测试 AppID 替换为正式微信小程序 AppID。
2. 把小程序 API 地址从本地地址替换为已备案的 HTTPS 域名，并加入微信服务器域名白名单。
3. 接入微信 `code2Session` 和手机号解密服务，替换当前开发身份头。
4. 配置 PostgreSQL、正式 JWT 密钥及对象存储；按照 `.env.example` 建立环境变量。
5. 录入正式馆型、器械、培训课程、岗位、推广任务与匹配规则。
